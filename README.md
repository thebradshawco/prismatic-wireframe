## Overview
This project is a generative 3D composition built with **Three.js**. It explores the intersection of technical wireframes and liminal space aesthetics. The execution is mathematically generated without external 3D assets, ensuring high performance and a lightweight footprint.

## Technical Features
* **Custom BufferGeometry:** The diamond is procedurally generated using trigonometric functions.
* **Aspect Ratio Optimized:** Hardcoded for a **4:5 vertical ratio**, ensuring safe-zone compatibility for 3:4 social media grid crops.
* **Infinite Loop:** A 15-second animation cycle designed for seamless video exports or background hero sections.
* **Zero Dependencies:** Uses a CDN-linked Three.js module for instant deployment.

## Usage
1. Clone the repository.
2. Open `index.html` in any modern web browser.
3. To adjust the movement speed, modify the `duration` variable in the animation loop.
